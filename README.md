# 🎓 Student Performance Prediction System

## 📌 Objective
Predict whether a student will pass or fail based on:
- Study Hours
- Attendance
- Internal Marks

## 🧠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## ⚙️ Machine Learning Model
- Logistic Regression

## 📊 Features
- Data preprocessing
- Model training & testing
- Accuracy evaluation
- Prediction for new student

## 🚀 How to Run
1. Install libraries:
   pip install pandas numpy matplotlib scikit-learn

2. Run:
   python student_performance.py

## 📈 Output
- Prediction (Pass/Fail)
- Accuracy Score
- Graph visualization
